### Javascript

Linguagem de programação.
Fracamente tipada.

```js
const nome = "João";
const age = 22;
```

### Typescript

Ainda é Javascript. Mas tem supersets. (Traz um conjunto de ferramentas para a pessoa desenvolvedora). Ou seja, traz na linguagem, formas de tipar variáveis.

```ts
const name: string = "João";
const age: number = 22;
```
